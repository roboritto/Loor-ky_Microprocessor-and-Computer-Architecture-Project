<H1>Loor-ky</H1>
A Microprocessor and Computer Architecture Project. It is a Digital Doorlock System that utilize the use of STM32F401RE Board, a 0.91" oled screen, 4x4 keypad, active buzzer and LEDs.
To ensure that the code work beautifully, this library (https://os.mbed.com/users/daniel_davvid/code/SSD1306_I2C/) has to be added to the project. To add a library, go to File>Add Mbed Library to Active Project, then paste the link.
